# PetsLog · 宠物健康手帐

多宠家庭的健康记录小程序：记录体重、就医、驱虫、疫苗、绝育等，管理主粮与宠物档案。

---

## 关于这个项目

**这是一份个人学习项目。**

本项目从零到可运行的小程序，**全程由 AI 辅助完成，本人未手写一行代码**。目的是通过「用自然语言描述需求 → AI 生成代码 → 调试与迭代」的方式，学习小程序与云开发的结构与流程，并验证 AI 编程在实际项目中的可用性。

如果你也在用 AI 做 side project，欢迎交流。

---

## 技术栈

| 类型     | 说明 |
|----------|------|
| 前端     | uni-app（Vue 3）+ 微信小程序 |
| 后端     | uniCloud（阿里云）：云函数 + 云数据库 + 云存储 |
| 功能要点 | 家庭/成员、宠物档案、事件记录、主粮记录、体重趋势展示等 |

---

## 功能概览

- **首页**：家庭名称可编辑、体重趋势（近 1 年）、宠物速览、当前/历史主粮，支持添加主粮
- **添加记录**：多宠批量录入，事件类型含称重、就医、驱虫、绝育、疫苗等，支持附件上传
- **历史**：记录列表，可编辑/删除
- **宠物档案**：宠物列表与详情，支持头像、品种、生日、病史、备注等

---

## 本地运行

1. 使用 [HBuilderX](https://www.dcloud.io/hbuilderx.html) 打开本项目，并绑定自己的 uniCloud 阿里云服务空间。
2. 在 uniCloud 控制台完成：云函数上传、DB Schema 上传、云存储目录创建（如 `pets`、`record`）。
3. 在 HBuilderX 中运行到「微信开发者工具」进行预览与调试。
